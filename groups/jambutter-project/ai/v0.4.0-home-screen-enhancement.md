## 🎯 What's New in v0.4.0

### **Home Screen Enhancements**

This release transforms the home screen from a static welcome page into an **action-oriented practice dashboard**.

### **✨ New Features**

#### **1. Quick Start Practice Buttons**
Three prominent buttons for instant practice sessions:
- **5 Min Warmup** - Yellow/butter gradient
- **15 Min Practice** - Toast brown gradient
- **30 Min Session** - Red/jam gradient

**How it works**:
- User clicks button → navigates to timer → automatically starts with selected duration
- Voice prompt "Let's go! You got this!" plays (if enabled)
- One-click to start practicing (down from 3+ clicks previously)

#### **2. Time-Based Greeting**
Buttery character displays contextual welcome messages based on time of day:
- **Morning (5am-12pm)**: "Good morning! Ready to practice?"
- **Afternoon (12pm-5pm)**: "Good afternoon! Let's make music!"
- **Evening (5pm-9pm)**: "Evening practice time!"
- **Night (9pm-5am)**: "Late night session? You're dedicated!"

Personalizes with user name if set in settings.

#### **3. Streak Display**
Current practice streak prominently displayed:
- Shows as "🔥 X Day Streak!" badge
- Red/jam color for visibility
- Positioned next to "Today's Progress" heading
- Motivates users to maintain consistency

#### **4. Enhanced Progress Card**
Today's progress now shows:
- Progress bar toward daily goal
- Minutes practiced / daily goal
- "🎉 Goal reached!" when complete
- Streak badge (if active)
